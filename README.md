# Algoritmo-Genetico
Trata-se de uma implementação em Python de um Algoritmo Genético aplicado à otimização de rotas em grafos, utilizando conceitos clássicos de computação evolutiva como seleção por roleta, cruzamento, mutação e elitismo para evoluir soluções ao longo das gerações.

O objetivo do algoritmo é encontrar uma rota de menor custo entre vértices de um grafo representado por matriz de adjacência, evitando repetições e minimizando o custo total do percurso.

No README estão descritos:

	•	conceitos utilizados
	•	estrutura do código
	•	parâmetros configuráveis

A implementação foi feita em Python com NumPy, com foco didático para demonstrar na prática o funcionamento de algoritmos genéticos em problemas de otimização combinatória.
